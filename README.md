# 🚢 潜水艇游戏 (Submarine Game)

## 📋 项目概述

这是一个使用C#/.NET开发的控制台潜水艇游戏，玩家需要在9x9的网格中寻找并击沉隐藏的潜水艇。

## 🎯 课题目标

### 基本要求
- ✅ **Board类实现** - 创建游戏界面显示类
- ✅ **输入处理** - 处理用户的坐标输入
- ✅ **潜水艇随机配置** - 在随机位置配置潜水艇
- ✅ **距离显示** - 显示与潜水艇的距离
- ✅ **游戏结束判定** - 检测是否击沉潜水艇
- ✅ **体验优化** - 提供良好的用户体验

## 🎮 游戏规则

### 基本规则
1. **游戏场地**: 9x9的网格 (A-I, 1-9)
2. **潜水艇配置**: 潜水艇随机隐藏在某个位置
3. **玩家目标**: 通过输入坐标来寻找并击沉潜水艇
4. **距离提示**: 每次输入后显示与潜水艇的距离
5. **游戏结束**: 找到潜水艇位置即获胜

### 游戏界面说明
```
 A B C D E F G H I  
1 | | | | | | | | | |
2 | | | | | | | | | |
3 | | | | | | | | | |
4 | | | | | | | | | |
5 | | | | | | | | | |
6 | | | | | | | | | |
7 | | | | | | | | | |
8 | | | | | | | | | |
9 | | | | | | | | | |
```

### 符号说明
- `|` - 未选择的位置
- `*` - 已选择但未击中的位置
- `X` - 击中潜水艇的位置

## 📝 类设计

### Board类
```csharp
public class Board
{
    // 游戏棋盘尺寸
    public int ROW_SIZE { get; set; }
    public int COLUMN_SIZE { get; set; }
    
    // 棋盘状态存储
    private CellState[,] cells;
    
    // 用户输入坐标
    private int inputRow;
    private int inputCol;
    
    // 潜水艇位置
    private int submarineRow;
    private int submarineCol;
    
    // 单元格状态枚举
    enum CellState
    {
        Unselected,  // 未选择
        Selected,    // 已选择
        Sunk         // 击沉
    }
}
```

### 主要方法
- `Print()` - 显示游戏棋盘
- `Input()` - 处理用户输入
- `InitializeSubmarine()` - 初始化潜水艇位置
- `CalculateDistance()` - 计算距离

## 🚀 使用方法

### 环境要求
1. 安装 .NET 8.0 SDK
2. 支持控制台的终端环境

### 运行步骤
1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd submarine
   ```

2. **编译运行**
   ```bash
   dotnet run
   ```

3. **游戏操作**
   - 输入行号 (1-9)
   - 输入列号 (A-I)
   - 查看距离提示
   - 继续寻找直到找到潜水艇
